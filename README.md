# さいひろみ：研究中・調査中の政策などについてまとめています。

ようこそ！このリポジトリは、さいひろみの政策や調査の政策などを皆さんと共に議論し、改善するためのオープンソースプロジェクトです。

## はじめに

わたし、さいひろみは、名取市内を歩いていて多くのお話や体験をしています。
その中で、改善したい、もっとよくしたいなどなど考えることがあります。
ただ、私ひとりの意見だけではなく、多くの皆さんの視点での確認や改善をしたら、もっと良くなるのでは？と考えています。
また、それを可視化できないかということで、GitHubの活用に至りました。
手探りなところが多いですが、少しずつ整備していきます。
